# express-backend

1. Test by command line **curl**:

- `/GET` : `curl url`
- `/POST` : `curl --data {query} url`
- `/DELETE` : `curl -X DELETE url`
- `/PUT` : `curl -X PUT -d {query} url`

2. Set up environment:
- Copy `.env.example` to `.env` and assign values.

3. Run with docker:
- Run command: `docker-compose up --build`
